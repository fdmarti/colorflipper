## Run Locally

Clone the project

```bash
  https://github.com/fdmarti/colorflipper.git
```

Go to the project directory

```bash
  cd colorflipper
```

Install dependencies

```bash
  npm i || npm install
```

Start the server

```bash
  npm run dev
```
